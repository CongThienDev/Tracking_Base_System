# Prompt Templates (Senior)

Use these templates to drive implementation of this repository end-to-end.

How to use:

1. Start with `00-master-system-execution.md`
2. Pick one focused prompt for the current sprint/task
3. Keep constraints and done criteria unchanged for predictable quality
4. Ask for concrete evidence output (commands, logs, screenshots, file diffs)
