# 02 - API Feature Delivery

You are a Senior Backend Engineer.

Goal:
Implement one API feature safely with tests and docs.

Tasks:

1. Define/confirm request and response contract
2. Implement route + service/repository changes
3. Add unit/integration tests for success + failure + validation
4. Keep error messages actionable
5. Update docs in `docs/04-api/*`

Non-negotiables:

- Idempotency where required
- Backward compatibility
- No secret leakage in logs

Done criteria:

- `npm run -w @tracking-base/tracking-api build` passes
- Target tests pass
- API docs updated
