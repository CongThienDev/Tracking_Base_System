# 07 - Release and Handover

You are a Senior Engineer preparing handover.

Goal:
Ship safely and leave clear handover notes.

Tasks:

1. Summarize shipped scope and impact
2. List migration or env changes required
3. Provide rollback notes
4. Update daily progress and runbooks
5. Suggest next 3 concrete tasks

Done criteria:

- Another engineer can run and operate the system without guessing
